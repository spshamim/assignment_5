Question-1: What are the different ways to select an element in the DOM?
✍️ There are several ways to select an element in the DOM. Some of the most common ways are: document.getElementById("id"); ➜ select an element by it's ID
document.getElementsByClassName("class"); ➜ select elements by class, it returns a HTMLCollection
document.getElementsByTagName("tag"); ➜ select elements by tag name, it returns a HTMLCollection
document.querySelector("selector"); ➜ select the first matching element, it used as CSS Selector
document.querySelectorAll("selector"); ➜ select all matching elements, it returns a NodeList

Question-3: What is event delegation in the DOM?
✍️ Event delegation is a technique in which we attach a single event listener to a parent element that will fire for all children matching a selector, whether they are present now or added in the future.
✍️ Useful for dynamically added elements.
✍️ It helps to improve performance by reducing the number of event listeners.
Question-4: What is event bubbling in the DOM?
✍️ Event bubbling is a mechanism in which an event is fired at the deepest (target) element first and then propagates upward to its ancestors (i.e. parents, grandparents etc.)
✍️ i.e. Clicking a button inside a div can trigger both the button's click event and the div's click event.
✍️ It is the default behavior of the browser.
✍️ It can be stopped using event.stopPropagation() method.

Question-5: How do you create, add, and remove elements using JavaScript?
✍️ To create, add, and remove elements using JavaScript, we can use the following methods: let parentElement = document.getElementById("parentID"); ➜ select the parent element
let element = document.createElement("tag"); ➜ create a new element with the specified tag name

parentElement.appendChild(element); ➜ add a new child element to the parent element
parentElement.insertBefore(newElement, referenceElement); ➜ insert a new child element before a reference element as a child of the parent element

parentElement.removeChild(element); ➜ remove a child element from the parent element
element.remove(); ➜ remove the element from the DOM
